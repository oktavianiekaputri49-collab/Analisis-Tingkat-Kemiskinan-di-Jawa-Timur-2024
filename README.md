# Analisis-Tingkat-Kemiskinan-di-Jawa-Timur-2024
Project ini membahas analisis tingkat kemiskinan kabupaten/kota di Jawa Timur tahun 2024 menggunakan data dari Badan Pusat Statistik (BPS). Analisis dilakukan untuk mengetahui perbedaan tingkat kemiskinan antar wilayah serta mengidentifikasi daerah dengan persentase kemiskinan tertinggi dan terendah.
